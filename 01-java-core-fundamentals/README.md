# 01 - Java Core Fundamentals ☕

Ce module couvre les bases fondamentales du langage Java 21. L'objectif est de passer d'un paradigme dynamique (Ruby) à un environnement statiquement typé et robuste.

## 📚 Programme du cours (OpenClassrooms)

### Partie 1 : Les bases de la syntaxe
* **Installation & Environnement** : Configuration du JDK 21 et de l'IDE.
* **Variables et Types** : Comprendre le typage statique (String, int, boolean, double).
* **Conditions & Boucles** : Structures de contrôle (if/else, switch, for, while).
* **Les Fonctions (Méthodes)** : Déclaration, paramètres et types de retour.

### Partie 2 : Programmation Orientée Objet (POO)
* **Classes et Objets** : Créer des modèles de données.
* **Propriétés et Méthodes** : Encapsulation (public, private, protected).
* **Constructeurs** : Initialisation des objets.
* **Héritage & Polymorphisme** : Réutilisation du code et spécialisation.

### Partie 3 : Structures de données avancées
* **Les Collections** : List (ArrayList), Set (HashSet), Map (HashMap).
* **Gestion des Exceptions** : Try/Catch et robustesse du code.
* **Streams & Lambdas (Java 8+)** : Manipulation moderne des données (équivalent des .map, .select de Ruby).

## 🚀 Objectif Technique
Réaliser le "Core" du projet **Epicrafter's Journey** : un moteur logique capable de gérer des blocs de construction sans interface graphique.

## 📝 Notes & Rappels (The Wagon ➔ Java)
* *Typage* : Contrairement à Ruby, chaque variable doit avoir un type défini.
* *Compilation* : Le code est transformé en Bytecode avant d'être exécuté par la JVM.
